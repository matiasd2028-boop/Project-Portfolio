Engineering & Academic Philosophy

I approach technology with a dual curiosity: building software systems that automate and predict, and assembling hardware components that physically interact with the environment. My academic journey through the International Baccalaureate (IB) program has instilled a rigorous, research-based framework into everything I develop.

Whether deploying high-altitude logistics payloads for weather balloons, building self-optimizing chess engines via python and API hooks, or structuring redundant server farms inside local network grids, I strive to overcome constraints using clean code, hardware optimization, and structured debugging.

My focus is on data integrity, embedded computing efficiency (Raspberry Pi/Arduino), and building real solutions that solve everyday challenges.
